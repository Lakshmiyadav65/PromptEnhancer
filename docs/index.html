PromptForge

Press a global hotkey anywhere on your computer to turn rough prompts into precise ones. Works in any app where you can select text.

Why?

Tools like Promptly AI work great as Chrome extensions for ChatGPT and Claude.ai. But they don't work inside the AI chats developers actually use — Claude Code in VS Code, Cursor's chat panel, BLACKBOX AI, locked-down webviews. PromptForge sidesteps that by operating at the OS level on whatever text you've selected.

Install

Download PromptForge_x64-setup.exe (~3 MB) from the latest release.
Run the installer. Windows SmartScreen will warn (we're not code-signed yet) — click "More info" → "Run anyway". This is normal for unsigned indie software.
Settings opens automatically on first launch. Click "Get a free Groq API key →", paste it back into Settings, click Save, then Test Connection.
You're done. Type a rough prompt anywhere, select it, press Ctrl + Alt + E.

Honest limitations

  • Windows only. macOS code paths exist but are untested.
  • Not code-signed. SmartScreen warning on first launch — see step 2 above.
  • No auto-install of updates. The app checks GitHub for newer releases but you'll re-run the installer manually.
  • Some apps block synthetic input. A handful of Electron-wrapped chat panels reject our synthetic Ctrl+C — you'll see a clear error if so. Notepad, Brave, VS Code's chat panels, and most native apps work.
  • The enhancement quality is set by the meta-prompt at prompts/enhancer-system-prompt.md — currently a placeholder. It works, but it'll get better as we calibrate it on real developer prompts. Send us a rough prompt you typed into Claude Code recently to help.
